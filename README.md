# Prayag Engineering

A professional, modern, and dynamic single-page website developed for Prayag Engineering.

## Features

- **Modern aesthetic**: Premium dark theme featuring smooth gradients, curated typography, and consistent branding.
- **Dynamic interaction**: Parallax scrolling effects and subtle micro-animations that enhance the user experience.
- **Responsive design**: Fully optimized and adaptable layout for all devices (mobile, tablet, desktop).
- **Functional contact form**: Registration-free FormSubmit.co integration for seamless email delivery.
- **Centralized configuration**: Easy maintenance of contact details, GST information, and service listings from a central configuration.

## Technologies Used

- **HTML5** for semantic structuring.
- **CSS3** (Vanilla) for custom styling, layouts, and animations.
- **JavaScript** for dynamic behaviors, form validation, and interactive UI elements.
